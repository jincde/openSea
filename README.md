### 아쉬운 점

- 통일된 화폐로고 svg 파일이 없어, 사이즈 차이가 발생
- 영상이 재생되는 배너가 사라져, 중간에 배너 레이아웃을 바꿨지만 다음엔 영상 배너 레이아웃을 구현할 것
- Swiper의 `prev/next`버튼 위치를 카드 테두리의 중간정도에 걸치게 할 것
- 데이터 파일을 만들어서 바인딩 하지 않고, 하드코딩한 점
